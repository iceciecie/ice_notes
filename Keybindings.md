#### vim
- show line number: **esc -> : -> set number**
- not show line number: **esc -> : -> set nonumber**
- skip to specified line: **esc -> : -> 100**

#### chrome
- new tab: **Ctrl + T**
- close tab: **Ctrl + W**
- skip tab: **Ctrl + 1 or Alt + 1**
- skip to address bar: **Ctrl + L**
- page down: **space**
- page up: **shift + place**
